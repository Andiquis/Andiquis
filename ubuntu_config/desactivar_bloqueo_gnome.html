🔓 Desactivar bloqueo automático de pantalla en GNOME

Este tutorial explica cómo evitar que GNOME bloquee la sesión cuando el sistema entra en inactividad. Ideal para equipos que:

  • deben seguir ejecutando procesos
  • no deben suspenderse
  • solo necesitan apagar visualmente la pantalla
---

🧠 Concepto clave

GNOME separa dos comportamientos distintos:

  • Inactividad → pantalla en negro / idle
  • Bloqueo → pedir contraseña

Aquí solo desactivamos el bloqueo, sin tocar:

  • suspensión
  • idle
  • ahorro de energía
---

🚫 Desactivar bloqueo automático

gsettings set org.gnome.desktop.screensaver lock-enabled false

✔ GNOME ya no bloqueará la sesión por inactividad.

---

🚫 Evitar bloqueo al suspender (opcional)

Aunque la suspensión esté desactivada, este ajuste evita bloqueos indirectos: 

gsettings set org.gnome.desktop.screensaver ubuntu-lock-on-suspend false
---

🔍 Verificar configuración

gsettings get org.gnome.desktop.screensaver lock-enabled

Resultado esperado: 

false
---

🧪 Comportamiento final

  • Inactividad → pantalla se apaga o baja brillo
  • ❌ No se bloquea
  • Movimiento → escritorio vuelve inmediato
  • Procesos siguen ejecutándose
  • Servidores siguen activos
---

⚠️ Nota de seguridad

Al desactivar el bloqueo automático, cualquier persona con acceso físico puede ver tu sesión activa.

Recomendado solo para:

  • equipos personales
  • entornos controlados
  • uso doméstico o laboratorio
